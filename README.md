# A website about my hiking adventures, built with Hugo and the Dream theme

I added several features to the theme:
- Waline comments (powered by Vercel). I chose to make the email optional and less strict spam by tweaking the front end and  adding some environmental variables in the vercel dashboard. The original module also has a url parsing issue, fixed by myself.
- Multiple languages (French, English and Chinese) plus a language switcher
- Pinnable posts
- Indicator of the last modified date
- More understandable icons
- Possibility to load image and mermaid diagrams in the summary
- Two themes: light and dark

For the moment, I'm quite happy with the theme and the features I added.
